# VirtualCollaboBaseとは

VR遠隔コラボレーションツールのベースとなるシステムです。

<img src=main.png width=70%>


OculusQuestで5人程度が同じルームに入っての会話をしたり、付箋やホワイトボードを使いアイデア出しなどが行えます。


ダウンロードして自由にアバターやステージ、機能などを変更・追加してください。

# Tested Environment
* Windows10
* Unity2018.3.8f1
* OculusQuest（Ver14）
